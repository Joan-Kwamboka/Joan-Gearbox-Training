Assignment 2

I built a geometry calculator using Object-Oriented Programming (OOP) concepts in Python. I was able to run a code that successfully calculated the area of a circle, square, rectangle and right angled triangle.From this assignment I was able to understand the concept of inheritance and polyporphism. The program demonstrated inheritance by creating a base class Shape and allowing other classes such as Circle, Square, Rectangle, and RightTriangle to inherit from it. These subclasses reuse the structure of the parent class while defining their own properties. Polymorphism was demonstrated by how each subclass overrides the area() method of the Shape class and implements it differently depending on the shape. When the area() method is called on different objects, Python executes the appropriate method for that specific shape, allowing the same method name to calculate different areas depending on the object.
